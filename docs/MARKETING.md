# DW Catalog WP

> **워드프레스에서 가장 유연한 상품 카탈로그 플러그인**  
> 하드코딩 없이, 코드 수정 없이 — 관리자 페이지에서 커스텀 필드를 자유롭게 설계하세요

---

## 왜 DW Catalog WP인가?

워드프레스 카탈로그 플러그인은 많지만, 대부분 **고정된 필드**만 제공합니다.  
상품명, 가격, 재고 — 그 이상 원하시면 개발자를 부르셔야 하죠.

**DW Catalog WP는 다릅니다.**  
식당 메뉴든, 장비 재고든, 책 컬렉션이든 — **필드를 직접 만들고, 수정하고, 정렬하세요.**

---

## 핵심 기능

### 커스텀 필드를 드래그 앤 드롭으로 관리
코드 한 줄 없이 관리자 페이지에서:
- **추가** — 원하는 필드를 언제든지 추가
- **편집** — 라벨, 타입, 필수 여부 변경
- **정렬** — 드래그로 순서 조정
- **삭제** — 불필요한 필드 즉시 제거

### 7가지 필드 타입 지원
| 타입 | 용도 |
|------|------|
| **Text** | 상품명, 브랜드, 일반 텍스트 |
| **Textarea** | 상세 설명, 내부 노트 |
| **Select** | 상태, 카테고리, 드롭다운 |
| **Number** | 가격, 수량, 무게 |
| **Email** | 문의처, 판매자 이메일 |
| **URL** | 제품 페이지, 레퍼런스 |
| **Date** | 출시일, 유통기한 |

### 엑셀/CSV 대량 업로드
- 수백 개 상품을 CSV 한 번으로 업로드
- 이미지 URL만 넣으면 자동 다운로드 → 미디어 라이브러리 등록
- 카테고리 자동 생성 (없으면 새로 만듦)
- 중복 건 자동 스킵

### 전문가 수준의 PDF 카탈로그 출력
- US Letter 사이즈, 카드 그리드 레이아웃
- **행당 1~4장** 자유 선택
- **원하는 필드만 골라서** 출력
- 카테고리별 필터링
- 제품 이미지 포함

### 관리자 목록 페이지 커스터마이징
- "List에 표시" 체크박스 하나로 컬럼 관리
- 60x60 썸네일 미리보기
- 상품명·품목코드 실시간 검색
- 정렬 가능한 컬럼

## 이런 분들을 위해 만들어졌습니다

### 식품 유통업체
- 품목코드, 팩 사이즈, 원산지, 상태 관리
- 정기적 카탈로그 PDF 출력
- CSV로 대량 가격 업데이트

### 장비 대여업체
- 모델명, 제조사, 임대료 관리
- 재고 상태 (Active / Out of Stock) 추적
- 고객용 카탈로그 PDF 제공

### 서점·도서관
- ISBN, 저자, 출판사, 장르
- 분류별 목록 관리
- 사내 재고 리포트

### 음식점
- 메뉴명, 가격, 알레르기 정보
- 카테고리별 (전채/메인/디저트) 분류
- 메뉴판 PDF 자동 생성

### 어떤 업종이든
**필드를 당신이 정의하세요.** 플러그인은 그대로 따라옵니다.

## 다른 플러그인과의 차이

| 기능 | DW Catalog WP | 일반 카탈로그 플러그인 | WooCommerce |
|------|:-------------:|:----------------------:|:-----------:|
| 동적 커스텀 필드 | ✅ | ❌ | ⚠️ (추가 플러그인 필요) |
| 드래그 정렬 필드 관리 | ✅ | ❌ | ❌ |
| CSV 대량 업로드 | ✅ | ⚠️ | ⚠️ (유료) |
| PDF 카탈로그 출력 | ✅ | ⚠️ (유료) | ❌ |
| 이미지 자동 다운로드 | ✅ | ❌ | ❌ |
| 코드 없이 필드 추가 | ✅ | ❌ | ❌ |
| 결제·장바구니 | ❌ | ⚠️ | ✅ |
| 라이선스 자동 업데이트 | ✅ | ⚠️ | ✅ |

**DW Catalog WP는 "판매"가 아닌 "카탈로그 관리"에 특화되어 있습니다.**  
장바구니·결제가 필요 없는 B2B, 재고 관리, 콘텐츠 포트폴리오에 최적.

## 설치 5분 — 운영 시작

### 1. 플러그인 업로드
`dw-catalog-wp-x.x.x.zip` 파일을 워드프레스에 업로드

### 2. 활성화
한 번 클릭

### 3. 커스텀 필드 정의
**DW Catalog > Custom Fields** 에서 원하는 필드 추가

### 4. 상품 등록
개별 등록 or CSV 일괄 업로드

### 5. PDF 출력
카테고리 선택 → 원하는 필드 체크 → PDF 생성

## 자주 묻는 질문

**Q. 기존 WooCommerce 데이터와 충돌하나요?**  
A. 아니요. 별도 포스트 타입으로 동작합니다.

**Q. 필드를 추가하면 기존 상품에도 적용되나요?**  
A. 네. 즉시 메타박스에 표시되고, 값을 입력하면 저장됩니다.

**Q. CSV 포맷이 정해져 있나요?**  
A. 유연합니다. 컬럼 헤더를 meta key로 지정하면 됩니다.  
   Semicolon(`;`), Comma(`,`), Tab 중 선택.

**Q. 여러 사이트에 같은 플러그인을 쓸 수 있나요?**  
A. 네. 각 사이트마다 다른 포스트 타입 구조를 설정할 수 있습니다.

**Q. 코드 수정 없이 두 번째 포스트 타입을 추가할 수 있나요?**  
A. 현재 버전은 기본 `product` 포스트 타입이 코드로 정의되어 있습니다.  
   필드는 UI로 자유 관리. 필요 시 설정 가능하도록 로드맵에 있습니다.
