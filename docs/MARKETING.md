# DW Catalog WP

> **워드프레스에서 가장 유연한 상품 카탈로그 플러그인**  
> 필드도, 디자인도 — 코드 수정 없이 관리자 페이지에서.

---

## 왜 DW Catalog WP인가?

워드프레스 카탈로그 플러그인은 많지만, 대부분 **고정된 필드**와 **고정된 디자인**만 제공합니다.  
상품명, 가격, 재고 — 그 이상을 원하시면 개발자를 부르셔야 하죠.

**DW Catalog WP는 다릅니다.**  
식당 메뉴든, 장비 재고든, 책 컬렉션이든 —  
**필드를 직접 만들고, 색과 폰트도 직접 바꾸세요.**

---

## 핵심 기능 세 가지

### 1. 커스텀 필드를 드래그 앤 드롭으로
- **추가** — 원하는 필드를 언제든지
- **편집** — 라벨, 타입, 필수 여부
- **정렬** — 드래그로 순서 조정
- **삭제** — 불필요한 필드 즉시 제거

**7가지 필드 타입**: Text, Textarea, Select, Number, Email, URL, Date

### 2. 프론트엔드 숏코드 3종
숏코드 하나로 제품을 멋지게 표시:

**`[dw_catalog_grid]`** — 반응형 카탈로그 그리드
```
[dw_catalog_grid columns="4" per_page="12"]
```

**`[dw_catalog_carousel]`** — 자동 재생 캐로셀
```
[dw_catalog_carousel per_slide="3" autoplay="yes"]
```

**`[dw_catalog_magazine]`** — 매거진 스타일 상세 뷰
```
[dw_catalog_magazine position="bottom-right" overlay="dark"]
```
제품 이미지 위에 제목과 커스텀 필드가 겹쳐지는 고급 레이아웃.  
오버레이 위치: 좌상/우상/좌하/우하/중앙 5가지 선택.

### 3. 디자인을 관리자에서 제어 ⭐ NEW
**DW Catalog > Shortcode Design** 메뉴에서:

- 🎨 **컬러피커** — 색상을 클릭으로 선택
- 📐 **타이포그래피** — 폰트 크기, 굵기, 색상 조절
- 🖼️ **레이아웃** — 카드 radius, 그림자, 간격 설정
- 📱 **매거진 오버레이** — Dark/Light 모드 배경 색상 커스터마이징

**CSS 한 줄 수정 없이** 브랜드에 맞게 디자인 완성.

## 이런 분들을 위해 만들어졌습니다

| 업종 | 활용 |
|------|------|
| **식품 유통업체** | 품목코드, 팩 사이즈, 원산지, 상태 관리 + 정기 PDF 카탈로그 |
| **장비 대여업체** | 모델명, 제조사, 재고 상태, 고객용 카탈로그 PDF |
| **서점·도서관** | ISBN, 저자, 출판사, 장르별 목록 |
| **음식점** | 메뉴명, 가격, 알레르기 정보, 메뉴판 자동 생성 |
| **포트폴리오** | 작품 제목, 제작연도, 재료, 매거진 스타일 쇼케이스 |
| **B2B 상품 카탈로그** | 결제 없이 상품 정보만 — 장바구니 불필요 |

**어떤 업종이든 — 필드를 당신이 정의하세요. 플러그인은 그대로 따라옵니다.**

## 추가 기능

### 엑셀/CSV 대량 업로드
- 수백 개 상품을 CSV 한 번으로 업로드
- **이미지 URL만 넣으면 자동 다운로드** → 미디어 라이브러리 자동 등록
- 카테고리 자동 생성 (없으면 새로 만듦)
- 중복 건 자동 스킵

### 전문가 수준의 PDF 카탈로그
- US Letter 사이즈, 카드 그리드 레이아웃
- **행당 1~4장** 자유 선택
- **원하는 필드만** 골라서 출력
- 카테고리별 필터링
- 제품 이미지 포함

### 관리자 목록 페이지 커스터마이징
- "List에 표시" 체크박스 하나로 컬럼 관리
- 60x60 썸네일 미리보기
- 상품명·품목코드 실시간 검색
- 정렬 가능한 컬럼

| 드래그 정렬 필드 관리 | ✅ | ❌ | ❌ |
| **숏코드 디자인 제어** | ✅ | ❌ | ❌ |
| 프론트엔드 그리드/캐로셀/매거진 | ✅ | ⚠️ | ⚠️ |
| CSV 대량 업로드 | ✅ | ⚠️ | ⚠️ (유료) |
| PDF 카탈로그 출력 | ✅ | ⚠️ (유료) | ❌ |
| 이미지 자동 다운로드 | ✅ | ❌ | ❌ |
| 코드 없이 필드/디자인 변경 | ✅ | ❌ | ❌ |
| 결제·장바구니 | ❌ | ⚠️ | ✅ |
| 라이선스 자동 업데이트 | ✅ | ⚠️ | ✅ |

**DW Catalog WP는 "판매"가 아닌 "카탈로그 관리"에 특화되어 있습니다.**  
장바구니·결제가 필요 없는 B2B, 재고 관리, 콘텐츠 포트폴리오에 최적.

## 설치 5분 — 운영 시작

### 1. 플러그인 업로드
`dw-catalog-wp-x.x.x.zip` 파일을 워드프레스에 업로드

### 2. 활성화
한 번 클릭

### 3. 커스텀 필드 정의
**DW Catalog > Custom Fields** 에서 원하는 필드 추가

### 4. 상품 등록
개별 등록 or CSV 일괄 업로드

### 5. 페이지에 숏코드 추가
```
[dw_catalog_grid columns="3"]
```

### 6. 디자인 커스터마이징
**DW Catalog > Shortcode Design** 에서 브랜드 컬러 & 폰트 설정

## 자주 묻는 질문

**Q. 기존 WooCommerce 데이터와 충돌하나요?**  
A. 아니요. 별도 포스트 타입(`product` with dw_pc_* prefix)으로 동작합니다.

**Q. 필드를 추가하면 기존 상품에도 적용되나요?**  
A. 네. 즉시 메타박스에 표시되고, 값을 입력하면 저장됩니다.

**Q. 디자인 설정을 바꾸면 기존 페이지에도 즉시 반영되나요?**  
A. 네. CSS 변수 기반이라 저장 즉시 모든 숏코드에 반영됩니다.  
   페이지 새로고침 한 번으로 완료.

**Q. CSV 포맷이 정해져 있나요?**  
A. 유연합니다. 컬럼 헤더를 meta key로 지정하면 됩니다.  
   Semicolon(`;`), Comma(`,`), Tab 중 선택.

**Q. 여러 사이트에 같은 플러그인을 쓸 수 있나요?**  
A. 네. 각 사이트마다 다른 커스텀 필드와 디자인을 설정할 수 있습니다.

**Q. 테마와 디자인 충돌이 없나요?**  
A. CSS 변수와 `!important`로 테마 오버라이드를 방지합니다.  
   관리자 설정이 우선 적용됩니다.

**Q. 매거진 스타일 상세 뷰는 어디에 쓰나요?**  
A. 제품 상세 페이지(single-post), 추천 상품 배너, 브랜드 스토리 섹션 등  
   시각적으로 강조하고 싶은 곳에 사용하세요.
